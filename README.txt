How to run:
    1. $ make (on terminal)
    2. $ ./socket_server (execute server)
    3. $ ./socket_client name (at most 11 characters) (execute client)

How to use:
    1. There will be two window name chatroom and read. Read window is for user to write message, chatroom window is for user to see the chatroom message.
    2. There are two special command for user to write message.
        1. /quit: quit client program.
        2. /sticker stickerID (look sticker.h to see the sticker id): send sticker.
    3. For normal message simply type what you want to send in read window.
